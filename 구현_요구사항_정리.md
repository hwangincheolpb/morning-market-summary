# 아침 시황 요약 자동화 - 구현 요구사항 정리

## 📋 프로젝트 목표
마감시황을 자동으로 수집 → 제미나이 젬스로 요약 → 나에게 발송

## 🎯 핵심 기능

### 1. 데이터 수집
- **텔레그램 채널 1개**: 마감시황 메시지 수집
- **웹사이트 1개**: 마감시황 페이지 수집
- **추가 사이트** (선택): 마감하자마자 올라오는 다른 마감시황 사이트

### 2. 요약 생성
- **Gemini API** 사용 (기존 GEM 프롬프트 활용)
- 간결 버전 양식: 3줄 요약 + 주요 지수 + 매크로 지표
- 상세 버전 양식 (선택): 표준 출력 양식 전체

### 3. 발송
- 나에게 발송
- 발송 방식: 텔레그램 봇 또는 문자 SMS (선택 필요)

## ❓ 확인 필요 사항

### 필수 정보
1. **텔레그램 채널**: 채널 이름/링크
2. **웹사이트**: 사이트 주소
3. **발송 방식**: 텔레그램 봇 / 문자 SMS / 둘 다
4. **실행 방식**: 자동(마감 후) / 수동 실행

### 선택 사항
5. **추가 마감시황 사이트**: 있으면 주소
6. **Gemini API 키**: 있으면 사용, 없으면 발급 필요
7. **텔레그램 봇 토큰**: 텔레그램 발송 시 필요
8. **문자 발송 서비스**: 문자 발송 시 (알리고 등)

## 🔧 기술 스택 (예상)

- **Python**: 메인 개발 언어
- **python-telegram-bot**: 텔레그램 채널/봇 연동
- **requests/BeautifulSoup**: 웹사이트 스크래핑
- **google-generativeai**: Gemini API 연동
- **schedule**: 자동 실행 스케줄링
