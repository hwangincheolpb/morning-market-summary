# 배포 후 할 일 (당신만 가능한 것)

아래 4가지만 하면 됩니다. 나머지는 이미 처리됨.

---

## 1. GitHub 레포 만들기 (없을 때만)

- https://github.com/new
- **Repository name**: `morning-market-summary`
- **Public** 또는 **Private** 선택
- **Add a README** 체크 해제 (빈 레포)
- **Create repository**

---

## 2. 푸시 (인증 1회)

**Cursor 터미널**에서:

```powershell
cd C:\Claudecode\아침_시황_요약_자동화
git push -u origin master
```

- 로그인/토큰 입력 창 나오면 처리
- GitHub 아이디가 **hwangincheolpb가 아니면**  
  `git remote set-url origin https://github.com/본인아이디/morning-market-summary.git`  
  로 바꾼 뒤 다시 `git push -u origin master`

---

## 3. Secrets 3개 등록 (한 번만)

레포 **Settings → Secrets and variables → Actions → New repository secret**

| Name | Value (직접 입력) |
|------|-------------------|
| `GEMINI_API_KEY` | Gemini API 키 |
| `TELEGRAM_BOT_TOKEN` | 텔레그램 봇 토큰 |
| `TELEGRAM_SEND_TO_CHAT_ID` | 받을 채팅 ID (숫자만, 예: 123456789) |

---

## 4. 워크플로 수동 실행 (테스트)

- **Actions** 탭 → **Daily Market Summary** → **Run workflow** (초록 버튼)
- 1–2분 뒤 **텔레그램(봇과 1:1 채팅)** 에 요약 도착

---

## 로컬 테스트 (선택)

`config.py` 열어서 3곳만 실제 값으로 바꾼 뒤:

```powershell
cd C:\Claudecode\아침_시황_요약_자동화
python main.py
```

→ 같은 **텔레그램 1:1 채팅** 으로 발송됨.
